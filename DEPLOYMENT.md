## Troubleshooting:

**Q: App not loading?**
- Wait 2-3 minutes after enabling GitHub Pages
- Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

**Q: Text-to-speech not working?**
- Use Chrome, Safari, or Edge (Firefox support varies)
- Make sure volume is up!

**Q: Images not loading?**
- Check internet connection
- Some schools/offices block Unsplash - that's OK, words still work!

**Q: API key not working?**
- Make sure you copied the entire key (no spaces)
- Check you have credit remaining at console.anthropic.com
